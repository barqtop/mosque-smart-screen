# 🕌 منظومة مواقيت الصلاة الذكية للمساجد (وقف خيري)

منصة رقمية وقفية مجانية بالكامل، مصممة خصيصاً لتشغيل شاشات مواقيت الصلاة الذكية داخل المساجد بكفاءة عالية ودون الحاجة لاتصال دائم بالإنترنت.

## ✨ مميزات المنظومة:
- **تشغيل محلي (Offline):** تعمل مباشرة من الفلاش ميموري (USB) عبر متصفح الشاشة الذكية.
- **دقة فلكية:** حسابات برمجية دقيقة للمواقيت مع إمكانية التعديل اليدوي لمطابقة التقويم الرسمي لبلدك.
- **واجهة مريحة:** تصميم متجاوب وثابت الأبعاد مخصص للشاشات الكبيرة في بيوت الله.

## 🛠️ طريقة التشغيل السريع:
1. قم بتحميل المشروع كملف مضغوط `ZIP`.
2. فك الضغط وانقل المجلد إلى فلاش ميموري (USB).
3. افتح ملف `control-panel.html` أولاً لضبط إعدادات مسجدك ومدينتك وحفظها.
4. قم بتشغيل ملف `index.html` على شاشة المسجد لتبدأ العرض الفوري.
